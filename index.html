About Me

An aspiring developer, I started my coding journey in 2022. I had no idea if I would enjoy it - or indeed be any good at it - but when I had a go at learning some html on W3Schools, I found it surprisingly fun!

Fast forward to now and I have learned the basics of html, CSS and Javascript, along with some C# thrown in for good measure. I've yet to decide which area to specialise in but the main thing for me at the moment is that I'm really enoying the learning process and creating my own content.

What I really didn't expect when I started coding was just how much I would enjoy it. Something that I'd previously found totally incomprehensible started to make sense! And the best bit is the satisfaction of writing code that works well and being able to say, "I made that!".
